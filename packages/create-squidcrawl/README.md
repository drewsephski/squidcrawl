# create-squidcrawl

Bootstrap and deploy a Squidcrawl stack from your terminal.

Status: early MVP (v0-only).

Deployment targets:

- Available now: `V0 API Worker only`
- Supporting soon: `Dashboard app + API`
- Supporting soon: `Fullstack app + auth + API`
- Supporting soon: `Custom domains and routes`

Quick start:

- Interactive location prompts: `npm create squidcrawl`
- Skip the location prompts: `npm create squidcrawl ../my-app`
- Local package dev mode: `pnpm -C packages/create-squidcrawl dev -- ../my-app`
